# NodeJS + React and Redux

A real-world React app that calls a remote API and authenticates users

## Feature List
* Sign Up 
* Login + Logout
* Form validations

## Technology Stack
### Server-side
- Express Framework
- Bookshelf ORM
- Knex SQL Query Builder
- PostgreSQL Database
- Token Based Authentication

### Client-side
- React + Redux + Router
- Bootstrap 4
- Lodash
- Webpack 1.x
  
Please enjoy. :rocket:
